# Kotlin Kullanıcı Listesi Uygulaması

Bu proje, JSONPlaceholder Users API üzerinden kullanıcı verilerini çekerek
MVVM mimarisi ile listeleyen bir Android uygulamasıdır.

## Kullanılan Teknolojiler
- Kotlin
- Jetpack Compose
- MVVM
- Retrofit
- Gson Converter
- Coroutines
- StateFlow
- Material 3

## Özellikler
- API'den kullanıcıları çekme
- Loading / Success / Error durum yönetimi
- LazyColumn ile kullanıcı listesi
- Card tabanlı modern arayüz

## API
https://jsonplaceholder.typicode.com/users
